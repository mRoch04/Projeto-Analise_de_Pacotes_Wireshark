# Análise de Pacotes de Rede com Wireshark

Este projeto apresenta uma análise básica de pacotes de rede utilizando o Wireshark, com foco em como o monitoramento do tráfego contribui para a cibersegurança e a mitigação de riscos.

## Tópicos Abordados
- Fundamentos de redes
- Captura de pacotes
- Análise com foco em segurança

## Nível
Iniciante / Cibersegurança (nível de entrada)
